# Quiz Dex

**Quiz-Dex** is a modern, responsive quiz web application built with **React**, **Vite**, **Tailwind CSS**, and **ShadCN UI components**. Challenge your knowledge across multiple categories, track your progress, and compete in daily challenges.

---

## Features

- Browse and search quiz categories
- Take quizzes with customizable difficulty and question count
- Track your stats: average score, best score, total quizzes taken
- Daily challenge with bonus points
- Recent activity tracking
- Fully responsive UI for desktop and mobile

---

## Tech Stack

- **Frontend:** React 18, Vite, TypeScript  
- **UI Components:** ShadCN, Radix UI, TailwindCSS  
- **State & Data:** React Query, Zod for validation  
- **Animations:** Framer Motion  
- **Charts & Stats:** Recharts  
- **Form Handling:** React Hook Form
